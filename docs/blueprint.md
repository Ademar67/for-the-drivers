# **App Name**: Liqui Moly Sales Hub

## Core Features:

- Clientes: Gestionar información de clientes.
- Seguimientos: Realizar seguimientos de clientes y prospectos.
- Pedidos: Gestionar pedidos de clientes.
- Facturas: Cobro de facturas
- Productos: Administrar el catálogo de productos de Liqui Moly.
- Mapa de Clientes: Visualizar la ubicación de los clientes en un mapa.
- Calculadora de Productos: Recomendar productos basados en las necesidades del cliente.
- Soporte IA: Asistente virtual para resolver dudas sobre los productos, uses a tool to determine when code snippets may be useful to display.
- Guía de Aceites: Información detallada sobre los aceites Liqui Moly.
- Guía de Aditivos: Información detallada sobre los aditivos Liqui Moly.

## Style Guidelines:

- Color primario: Azul medianoche (#00468E) para reflejar la identidad corporativa.
- Color secundario: Blanco (#FFFFFF) para claridad y limpieza.
- Color de acento: Rojo máximo (#DA251D) para resaltar acciones importantes.
- Tipografía: 'Inter', sans-serif, para títulos y cuerpo del texto, ofreciendo un aspecto moderno y limpio.
- Iconos de lucide-react para una interfaz intuitiva y profesional.
- Diseño de tarjetas en una cuadrícula de 3x3 para una presentación organizada y fácil de usar.
- Animaciones sutiles para mejorar la interactividad y la experiencia del usuario.
